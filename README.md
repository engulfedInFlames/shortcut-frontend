## 명령어

- `npm init` => venv + [requirements.txt]
- `npm init -y`
- `npm install` - node_modules = venv
- `npm i`

- 패키지 설치
  - `pip install package-name` => `npm i package-name`

## 패키지 설명

- babel
  : 최신 JS 문법 컴파일러
- nodemon
  : 프로젝트 변경 사항 감지
- morgan
  : request log 출력
